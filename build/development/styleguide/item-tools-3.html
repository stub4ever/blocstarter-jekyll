2.3 Helpers

  • .accessibility .js-accessibility - Used for hiding content. It's important that a technique be implemented that results in the desired outcome and accessibility
  • .group - Used in float layouts where elements are floated to be stacked horizontally (Clearfix)
  • .skip-to-main - Used to skip navigation link to content, pointing to an anchor link placed in the same page which allow the user to skip directly to the main content. Having a skip to main content link appear when it receives keyboard focus
