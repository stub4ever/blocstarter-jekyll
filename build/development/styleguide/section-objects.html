5 Objects

Object oriented CSS. Begin using classes exclusively for agnostically named noncosmetic design patterns. This layer is mostly layout-driven.

Source: styles/5-objects/_section.scss, line 1

5.1 Section

  • A wrapper element and can be used to give space between other section.
    It has by default an full-width, but for some case you can add fixed-with for example wide-screen at 1700px above.

Examples

Default styling

section__inner

A containing element to wrap elements inside a section. It has a fixed width and a half gutter to stack elements that placed with half gutter

<div class="section">
    <div class="section__inner"></div>
</div>

5.2 Box

  • A box element spacing scale based on powers of different base font-size / lineheight.
    By default it has a half gutter that stack with section__inner or core settings variable

Example

<div class="section">
    <div class="section__inner  section__inner--medium@tablet  section__inner--large@desktop">
        <div class="box  box--medium@tablet  box--large@desktop"></div>
    </div>
</div>

Source: styles/5-objects/_reduce.scss, line 1

5.3 Reduce

  • Add negative spacing on element to scale based on powers of different base font-size / lineheight.
    By default it has a negative half gutter that stack with section__inner or core settings variables

Example

<div class="section">
    <div class="section__inner  section__inner--medium@tablet">
        <div class="reduce  reduce@tablet">
            <div class="box"></div>
            <div class="box"></div>
        </div>
    </div>
</div>

5.4 Bloc

  • bloc float system.
    The bloc object provides us with a column-style bloc system.
    This file contains the basic structural elements, but classes should be complemented with width utilities

Example

<!--The above will create a system in which the first item will be 100% width until we enter our desktop breakpoint,
when it will become 33.333% width. The second and third items will be 50% of their parent, until they also become
33.333% width at the desktop breakpoint.
-->
<div class="bloc">
    <div class="bloc__item  w-1/2">
    </div>
    <div class="bloc__item  w-1/2">
    </div>
</div>

5.6 Crop

  • A list of cropping ratios that get generated as modifier classes.
    Provide a cropping container in order to display media (usually images)
    cropped to certain ratios.

A default list of cropping ratios that get generated as modifier classes.

  • 2:1
  • 4:3
  • 16:9

Example

<div class="crop  crop--2:1">
    <img class="crop__content crop__content--center" src="https://unsplash.it/1000/500" alt="">
</div>

5.7 Flag

  • The flag object is a design pattern similar to the media object, however it utilises display: table[-cell]; to give us control over the vertical alignments of the text and image.

http://csswizardry.com/2013/05/the-flag-object

Example

    </div>
</div>

    <img class="ratio__content" src="https://unsplash.it/1000/500" alt="">
</div>

  • The pack object simply causes any number of elements pack up horizontally to
    automatically fill an equal, fluid width of their parent.

Example
