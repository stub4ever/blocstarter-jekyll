Section
Block Element / Modifier What it does?
section A wrapper element and can be used to give space between other section. It has by default an full-width, but for some case you can add fixed-with for example wide-screen at 1700px above.
__inner A containing element to wrap elements inside a section. It has a fixed width and a half gutter to stack elements that placed with half gutter
Box
Block Element / Modifier What it does?
box A box element spacing scale based on powers of different base font-size / lineheight. By default it has a half gutter that stack with other elements and section__inner
